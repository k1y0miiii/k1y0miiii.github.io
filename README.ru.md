# k1y0miiii.github.io

[English](README.md) · **Русский**

Исходники моего личного сайта — **https://k1y0miiii.github.io**

Статический портфолио-сайт (HTML/CSS/JS, без сборки): кто я и избранные проекты по
AI/ML, инструментам для LLM, безопасности и системному коду, плюс короткий блок про
экономию токенов со ссылкой на живой демо [token-economics](https://k1y0miiii.github.io/token-economics/).

## Разработка

Откройте `index.html` напрямую или поднимите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Структура

- `index.html` — разметка
- `styles.css` — тёмная графит/лайм тема (Syne + JetBrains Mono)
- `app.js` — reveal-анимации, счётчики, фильтр проектов
