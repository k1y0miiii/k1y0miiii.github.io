# k1y0miiii.github.io

**English** · [Русский](README.ru.md)

Source for my personal site — **https://k1y0miiii.github.io**

A static developer portfolio (HTML/CSS/JS, no build step): who I am and selected work
across AI/ML, LLM tooling, security and systems, plus a short token-efficiency section
that links to the live [token-economics](https://k1y0miiii.github.io/token-economics/) demo.

## Develop

Open `index.html` directly, or serve locally:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Structure

- `index.html` — markup
- `styles.css` — dark graphite/lime theme (Syne + JetBrains Mono)
- `app.js` — scroll reveals, count-ups, project filter
