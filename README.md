## Stack

Vanilla HTML / CSS / JS. No frameworks, no bundler.

- `index.html` — content + EN markup (RU strings live in `app.js`)
- `styles.css` — design system: dark + lime, JetBrains Mono / Syne
- `app.js` — i18n switcher, typed hero, canvas token field, custom cursor, context-window HUD

Run locally with `python3 -m http.server`.
